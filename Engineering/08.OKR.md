# OKR 

Objectives and key results é uma metodologia utilizada para definir objetivos e medir resultados dentro de equipes e organizações.

OKR significa:

- O → Objectives;
- KR → Key Results.

## Estrutura do OKR

### Objective

O objetivo representa:

- o que se deseja alcançar;
- a direção estratégica;
- o propósito da meta.

Ele deve ser:

- claro;
- inspirador;
- qualitativo;
- motivador.

Exemplo:

```txt
"Melhorar a satisfação do cliente"
```

### Key Results

Os Key Results definem:

- como o sucesso será medido;
- quais métricas indicam progresso;
- quais resultados precisam ser atingidos.

Eles devem ser:

- quantitativos;
- específicos;
- mensuráveis;
- desafiadores.

Exemplo:

- Atingir 90% de satisfação;
- Reduzir tempo de resposta para 2 horas;
- Aumentar retenção em 15%.

## Como funciona?

Os OKRs normalmente funcionam em ciclos:

- trimestrais;
- semestrais;
- anuais.

Durante o ciclo:

- equipes acompanham progresso;
- revisam métricas;
- ajustam prioridades;
- avaliam resultados.

## Benefícios

- Foco;
- Alinhamento;
- Transparência;
- Clareza de objetivos;
- Motivação;
- Acompanhamento contínuo.


## Características importantes

- Objetivos ambiciosos;
- Resultados mensuráveis;
- Transparência organizacional;
- Revisão frequente;
- Cultura orientada a resultados.

## Exemplo prático

Empresa SaaS

```txt
Objetivo: Melhorar a experiência do usuário
```
Key Results
- Reduzir churn em 10%;
- Aumentar tempo médio de uso;
- Obter 85% de satisfação.

## OKR vs KPI

|           OKR         |           KPI          |
|          :---:        |          :---:         |
|     Define direção    |     Mede performance   |
|   Mais estratégico 	  |     Mais operacional   |
| Objetivos ambiciosos  |  Indicadores contínuos |
