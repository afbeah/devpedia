# LLM (Large Language Model)

LLM (Large Language Model) é um modelo de Inteligência Artificial treinado para compreender, gerar e manipular linguagem natural.

Seu principal objetivo é interpretar textos e produzir respostas semelhantes às que seriam escritas por um ser humano.

Em outras palavras:

> Um LLM é o "cérebro" por trás de aplicações como ChatGPT, Gemini, Claude e Copilot.

## O que significa LLM?

A sigla significa:

```txt
Large Language Model
```

Em português:

```txt
Modelo de Linguagem de Grande Escala
```

O termo "grande escala" refere-se à enorme quantidade de dados utilizada durante o treinamento e ao número de parâmetros presentes no modelo.

## O problema

Imagine que você deseja criar um sistema capaz de:

* responder perguntas;
* resumir textos;
* traduzir idiomas;
* gerar código;
* escrever documentos.

Criar regras para todas essas tarefas seria praticamente impossível.

Os LLMs resolvem esse problema aprendendo padrões presentes em bilhões de exemplos de texto.

## Como funciona?

Durante o treinamento, o modelo aprende relações entre palavras, frases e conceitos.

De forma simplificada:

```txt
Grande volume de textos
↓
Treinamento
↓
LLM
↓
Capacidade de gerar respostas
```

O modelo não memoriza respostas prontas.

Ele aprende padrões estatísticos da linguagem para prever a próxima palavra mais provável em uma sequência.

## Exemplo

Pergunta:

```txt
Qual é a capital da França?
```

Resposta:

```txt
Paris.
```

Para o usuário parece simples.

Internamente, o modelo calcula qual sequência de palavras possui maior probabilidade de responder corretamente à pergunta.

## Capacidades

Um LLM pode realizar tarefas como:

* responder perguntas;
* resumir documentos;
* traduzir idiomas;
* gerar código;
* explicar conceitos;
* criar conteúdos;
* auxiliar em análises.

Tudo isso utilizando linguagem natural.

## Limitações

Apesar de muito poderoso, um LLM possui limitações importantes.

### Conhecimento limitado

O modelo conhece apenas as informações presentes durante seu treinamento.

Não possui acesso automático a dados recentes ou privados.

### Alucinações

Em alguns casos o modelo pode gerar respostas incorretas apresentadas com confiança.

Esse fenômeno é conhecido como:

```txt
Hallucination
```

### Contexto limitado

Cada modelo possui uma janela de contexto (Context Window).

Se a quantidade de informações ultrapassar esse limite, parte do contexto deixa de ser considerada.

## Exemplos de LLMs

Alguns dos principais modelos disponíveis atualmente:

* GPT (OpenAI)
* Gemini (Google)
* Claude (Anthropic)
* Llama (Meta)
* Mistral

Cada um possui características e capacidades específicas.

## Como aplicações utilizam LLMs?

Uma aplicação típica segue o fluxo:

```txt
Usuário
↓
Prompt
↓
LLM
↓
Resposta
```

Esse é o modelo mais simples de utilização.

## O problema do modelo simples

Imagine uma empresa que deseja responder perguntas utilizando documentos internos.

Fluxo:

```txt
Usuário
↓
LLM
↓
Resposta
```

O modelo não possui acesso aos documentos da empresa.

Como consequência, pode responder de forma incompleta ou incorreta.

Foi justamente para resolver esse problema que surgiu uma arquitetura chamada:

```txt
RAG
(Retrieval-Augmented Generation)
```

## Relação com outros conceitos

Fluxo de aprendizado:

```txt
LLM

↓
RAG

Embeddings
↓
Vector Database

```

Esses conceitos representam a base da Engenharia de IA moderna.

## Resumo

LLM (Large Language Model) é um modelo de Inteligência Artificial treinado para compreender e gerar linguagem natural.

Ele é a base de aplicações como ChatGPT, Gemini e Claude, permitindo responder perguntas, gerar código, resumir documentos e executar diversas tarefas relacionadas à linguagem. Entretanto, suas limitações em relação ao conhecimento atualizado e ao acesso a informações externas motivaram o surgimento de arquiteturas como o RAG.
