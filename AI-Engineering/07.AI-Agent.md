# AI Agent

Um AI Agent (Agente de Inteligência Artificial) é um sistema capaz de perceber um objetivo, tomar decisões, planejar ações, utilizar ferramentas e executar tarefas de forma autônoma para alcançar um resultado.

Seu principal objetivo é permitir que modelos de Inteligência Artificial resolvam problemas complexos que exigem múltiplas etapas.

Em outras palavras:

> Um AI Agent transforma um LLM em um solucionador de problemas.

## O problema

Imagine que um usuário faça a seguinte solicitação:

```txt
Analise meu currículo, procure vagas compatíveis, adapte meu currículo para cada vaga e gere uma carta de apresentação.
```

Essa tarefa envolve diversas etapas:

- analisar o currículo;
- identificar competências;
- pesquisar vagas;
- comparar requisitos;
- adaptar o currículo;
- gerar uma carta de apresentação.

Um LLM tradicional consegue responder perguntas.

Mas ele não sabe coordenar automaticamente todas essas tarefas.

## Como funciona?

Um AI Agent atua como um orquestrador.

Fluxo:

```txt
Usuário
↓
Objetivo
↓
AI Agent
├── Planejamento
├── Tomada de decisão
├── Escolha de ferramentas
└── Execução
↓
Resposta
```

O agente divide o problema em pequenas tarefas.

Depois decide como resolvê-las.

## Exemplo

Pedido do usuário:

```txt
Quero reservar uma viagem para São Paulo.
```

Fluxo:

```txt
Usuário
↓
AI Agent
↓
Pesquisar voos
↓
Pesquisar hotéis
↓
Comparar preços
↓
Selecionar melhores opções
↓
Gerar resposta
```

Perceba que o usuário realizou apenas um pedido.

O agente executou diversas tarefas internamente.

## Componentes de um AI Agent

Normalmente um agente possui:

### LLM

Responsável pelo raciocínio.

### Memória

Armazena informações durante a execução.

### Ferramentas

Permitem acessar APIs, bancos de dados e outros sistemas.

### Planejamento

Define quais etapas serão executadas.

### Tomada de decisão

Escolhe qual ação executar em cada momento.

## AI Agent x LLM

Embora estejam relacionados, não são a mesma coisa.

| LLM | AI Agent |
|------|----------|
| Gera texto | Resolve problemas |
| Responde perguntas | Executa tarefas |
| Atua em uma única interação | Pode realizar múltiplas etapas |
| Não utiliza ferramentas sozinho | Utiliza ferramentas quando necessário |

## AI Agent x Function Calling

Function Calling permite executar uma única função.

Exemplo:

```txt
Consultar clima.
```

Um AI Agent pode executar diversas funções em sequência.

Exemplo:

```txt
Pesquisar voo
↓
Pesquisar hotel
↓
Consultar clima
↓
Montar roteiro
```

O Agent utiliza Function Calling como um de seus recursos.

## Benefícios

### Automação

Permite automatizar processos complexos.

### Planejamento

Divide problemas grandes em pequenas tarefas.

### Integração

Pode utilizar APIs, bancos de dados e serviços externos.

### Flexibilidade

Adapta suas ações conforme o contexto.

## Limitações

### Complexidade

Agentes são mais difíceis de desenvolver.

### Custos

Múltiplas chamadas ao LLM aumentam o consumo.

### Tempo de execução

Mais etapas significam maior tempo de processamento.

### Segurança

É necessário controlar cuidadosamente quais ferramentas o agente pode utilizar.

## Exemplos de uso

AI Agents são utilizados em:

- assistentes pessoais;
- copilotos de programação;
- automação empresarial;
- atendimento ao cliente;
- análise de documentos;
- planejamento de tarefas;
- pesquisa inteligente.

## Resumo

Um AI Agent é um sistema que utiliza um Large Language Model para planejar, tomar decisões e executar tarefas de forma autônoma.

Ao combinar LLMs, memória, ferramentas, RAG e Function Calling, os agentes tornam possível a construção de aplicações inteligentes capazes de resolver problemas complexos de maneira automatizada.
