# JSON

*JavaScript Object Notation* (JSON) é um formato leve utilizado para troca e armazenamento de dados estruturados.

Ele é amplamente utilizado na web por ser:

- simples de ler;
- fácil de escrever;
- leve;
- fácil de interpretar por máquinas.

JSON é muito usado na comunicação entre:

- APIs;
- servidores;
- aplicações web;
- aplicativos mobile;
- bancos de dados;
- microsserviços.

Normalmente os dados são enviados em formato de texto.

## Exemplo Simples de JSON

```JSON
{
  "nome": "Beatriz",
  "idade": 33,
  "tecnologias": ["Go", "React", "TypeScript"]
}
```

## Onde o JSON é usado?

- APIs REST;
- Configurações de aplicações;
- Respostas de servidores;
- Armazenamento de dados;
- Comunicação entre sistemas.
