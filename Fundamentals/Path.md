# Path

O termo pode ter diferentes significados dependendo do contexto.

1. Path em URL (rotas de API) também conhecido como endpoint path ou route path.
   
   É a parte do endereço que normalmente associado a recursos REST.
   ex:
     https://api.exemplo.com/**users/123**

   O Path é **/users/123**, indicando que estamos acessando o usuário com ID 123.

2. Path em sistemas de arquivos

   É o caminho para localizar um arquivo ou diretório dentro do sistema.
   ex:
     /home/**user**/documentos/**arquivo.txt** (Linux, macOs)
     C:\Users\nome\Documents\arquivo.txt

3. Path em programação (importação de módulos e recursos)

   Em algumas liguagens, o path é usado para importar arquivos ou pacotes.
   ex:
     import sys
     sys.path.append("/meu/diretorio/personalizado")
