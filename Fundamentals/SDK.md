# SDK 

*SDK - Software Development Kit* é um conjunto de ferramentas e recursos fornecidos por empresas ou plataformas para ajudar outros programadores a criar software para uma plataforma. 

> É um conjunto de ferramentas e recursos que desenvolvedores usam para criar aplicativos ou outros software. Pense nela como uma "caixa de ferramentas" que contém tudo o que você precisa para construir algo, como códigos prontos, bibliotecas, documentações e até exemplos de como usar essas ferramentas.

## Principais Componentes de um SDK

- Bibliotecas/APIs: Conjuntos de funções e métodos que facilitam a interação com os serviços ou recursos da plataforma. São como pedaços de código prontos que podem ser usados em projetos, em vez de criar tudo do zero.
- Ferramentas: Utilitários que ajudam a compilar, depurar e testar o software. Imagine que quer construir uma casa, será necessário martelo, pregos, madeira, etc. Em desenvolvimento de software esses "martelos" e "pregos" são as ferramentas que os desenvolvedores usam para escrever código, testar o aplicativo e garantir que tudo funciona corretamente.
- Documentação: Guias e referências para ajudar os devs a entender como usar as bibliotecas e ferramentas. Instruções que explicam como usar as ferramentas e bibliotecas.

## Exemplos de SDKs populares

- Android SDK;
- AWS SDK;
- Firebase SDK;
- Stripe SDK;
- OpenAI SDK;
- Google Maps SDK.

## Diferença entre SDK e API

- API → interface de comunicação entre sistemas;
- SDK → conjunto completo de ferramentas para desenvolvimento.

## SDK de TV

É um conjunto de ferramentas e recursos oferecidos para desenvolvedores criarem aplicativos e serviços para plataformas de SmartTVs. Assim como outros SDKs, ele facilita o desenvolvimento, fornecendo ao desenvolvedor tudo o que precisa para construir, testar e implementar aplicativos para TVs.

É um kit de desenvolvimento específico para criar aplicativos ou funcionalidades para TVs inteligentes. Com ela os desenvolvedores podem criar aplicativos que rodam diretamente na TV, como streaming ou jogos.
