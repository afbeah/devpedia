# Landing Page

## "A PORTA DE ENTRADA QUE CONVERTE" 

Uma **Landing Page** (ou página de aterrisagem) é uma página web super focada, criada com um único objetivo: **Converter o visitante a tomar uma ação específica**. 
> ex: pode ser um formulário de cadastro, download de um app, compra de um produto ou até mesmo assinar uma newsletter.


### Para que serve?
- Conversão: Transformar visitantes em leads/clientes
- Segmentação: Direcionar campanhas de ads
  > ex: anúncio do instagram leva para uma landing page só sobre promoção X.
- Testes: Testar diferentes versões (cores, textos, botões) para ver o que converte mais (teste a/b).

### Como Funciona?

1. Visita: O usuário chega na página
 > via link de anúncio, e-mail, rede social

2. Engajamento: A página tem um design limpo, sem distrações (sem menu, links externos), só o essencial:
 - Titúlo chamativo
  > "Ganhe 50% off hoje!"
 - Benefícios claros
  > "Aprenda Python em 30 dias"
 - Call to Action (CTA)
  > Botão do tipo: "Quero desconto!"

3. Conversão: O usuário preenche um formulário, clica no botão e sucesso.

### O que não é uma Landing Page?

- Site institucional
 > tem várias páginas, menu, infos gerais
- Blog
 > foco é conteúdo, não conversão
