# Robo 3T

É um cliente gráfico para o MongoDB. Ele permite visualizar, consultar e manipular bancos de dados MongoDB de forma interativa, de forma semelhante ao que o *PGAdmin* faz para o *PostgreSQL* 

## O que é o Robo 3T?

O **Robo 3T** é uma interface gráfica (*GUI*) para o MongoDB, permitindo que:

- Conecte-se a um servidor MongoDB local ou remoto;
- Explore bancos de dados e coleções de forma visual;
- Execute consultas MongoDB Query Language (*MQL*);
- Insira, edite e exclua documentos;
- Exporte e importe dados;
- Rodar scripts em JavaScript diretamente no Shell do MongoDB.

## Principais Funcionalidades

- Exploração visual: Você pode navegar pelos bancos de dados e coleções com facilidade;
- Execução de consultas: Pode executar consultas baseadas em documentos JSON-like;
- Edição de Documentos: Clique duas vezes em um campo para editar diretamente;
- Shell Embutido: Permite rodar comandos MongoDB diretamente, semelhante ao terminal.

## Conclusão

O **ROBO 3T** é uma ferramenta muito útil para quem trabalha com MongoDB, pois facilita a visualização e manipulação dos dados. Ele é leve, rápido e permite interações sem precisar escrever comandos no terminal.
