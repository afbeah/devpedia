# Prometheus

É uma ferramenta de monitoramento e alertas de código aberto desenvolvida originalmente pela SoundCloud. É amplamente utilizada para coletar e armazenar métricas de tempo real em um banco de dados de séries temporais, permitindo consultas flexíveis e alertas baseados nessas métricas. No contexto de Go, o Prometheus é frequentemente utilizado para monitorar aplicações, aproveitando a biblioteca *client_golang* para expor métricas customizadas. 

Integrar o Prometheus em uma aplicação Go é uma maneira eficaz de monitorar e melhorar a performance, proporcionando insights valiosos sobre a aplicação.

## O que o Prometheus monitora?

- Uso de CPU;
- Memória;
- Latência;
- Número de requisições;
- Erros;
- Tempo de resposta;
- Containers e Kubernetes.
